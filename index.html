Run the assistant from this deployed dashboard. Install the connector userscript to bridge this page with a Jackbox browser tab that reads prompts, asks WebLLM for a local answer, and submits it for you.

Workflow

Install, connect, play

  1. Install Tampermonkey or Greasemonkey. The connector is a userscript, so it needs a userscript manager.
  2. Install this connector: jackbox-ai-connector.user.js.
  3. Keep this dashboard open. The connector also runs here and relays commands to Jackbox.
  4. Enter a room code and launch bot tabs. Each Jackbox tab is one autonomous player session.
  5. Click Join all bots. The dashboard routes names/config to each connected tab.
  6. Keep the dashboard open. Prompts are relayed here, answered with WebLLM, and sent back to each tab.
Important: this static edition controls the assistant from the deployed page, then automates the actual Jackbox web page through a userscript running in the Jackbox tab. It does not reverse-engineer private Jackbox APIs and does not require a Python/Flask/Selenium server.
