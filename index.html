Jackbox AI Assistant

Run the assistant as a static website. Install the connector userscript to let a Jackbox browser tab read prompts, ask this page for a local answer, and submit it for you.

Install connector Setup workflow

Workflow

Install, connect, play

  1. Install Tampermonkey or Greasemonkey. The connector is a userscript, so it needs a userscript manager.
  2. Install this connector: jackbox-ai-connector.user.js.
  3. Open jackbox.tv. The connector injects a compact AI panel into the page.
  4. Select a persona and enable automation. One Jackbox tab represents one autonomous AI player.
  5. Use more tabs for more bots. Each tab keeps its own persona, room, username, and logs.
Important: this static edition automates the actual Jackbox web page from a userscript. It does not reverse-engineer private Jackbox APIs and does not require a Python/Flask/Selenium server.
